# VoIP.ms SMS Web

This is a web-based port of the [VoIP.ms SMS Android application](https://github.com/michaelkourlas/voipms-sms-client). It is an ethical LLM-assisted port designed to provide a similar experience on the web.

## Overview

VoIP.ms SMS Web is a messaging application for VoIP.ms that seeks to replicate the aesthetic and functionality of the original Android app in a web environment.

## Features

- Modern web interface (Next.js, Tailwind CSS)
- Synchronization with VoIP.ms
- Progressive Web App (PWA) support
- Responsive design for mobile and desktop

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
